# 소아벨라 이미지 alt 텍스트 작성 가이드

> Cafe24 관리자 → 상품 관리 → 상품 목록 → 각 상품 편집 → 이미지 alt 입력란

---

## alt 텍스트가 중요한 이유

1. **이미지 검색 유입**: 구글/네이버 이미지 검색에서 "금반지", "순금목걸이" 검색 시 노출
2. **크롤러 이해**: 검색엔진은 이미지를 보지 못하고 alt 텍스트를 읽음
3. **접근성**: 시각장애인 스크린리더가 alt 텍스트를 읽어줌 (웹 접근성 법적 요건)

---

## 작성 공식

```
[소재/순도] + [제품 종류] + [특징/디자인] + [용도/상황] (선택)
```

### 예시

| 상품명 | 나쁜 alt | 좋은 alt |
|--------|---------|---------|
| 24k 순금 슈가 하트 반지 1.875g | `""` (없음) | `소아벨라 24K 순금반지 하트 디자인 1.875g — 돌잔치 선물 추천` |
| 24k 순금 퓨어 리본 목걸이 3.75g | `rb0506.jpg` | `소아벨라 24K 순금목걸이 리본 펜던트 3.75g — 결혼기념일 선물` |
| 14k 18k 럭스 링크 큐빅 목걸이 | `상품이미지` | `소아벨라 14K 18K 골드 큐빅 링크 목걸이 — 레이어드 스타일링` |
| 14k 18k 러프 하트 레이스 반지 | `금반지` | `소아벨라 18K 골드 하트 레이스 반지 — 커플링 약혼반지 추천` |
| 24k 순금 커플링 세트 | `커플반지` | `소아벨라 24K 순금 커플링 세트 — 프러포즈 결혼 기념반지` |

## 카테고리별 alt 키워드 패턴

### 반지 (cate_no=25)
```
소아벨라 [24K 순금 / 14K / 18K] 반지 [디자인명] — [돌반지 / 커플링 / 약혼반지 / 기념반지]
```
**노려야 할 검색어**: 금반지, 순금반지, 돌반지, 커플링, 14k반지, 18k반지

### 목걸이 (cate_no=27)
```
소아벨라 [24K 순금 / 14K / 18K] 목걸이 [디자인명] — [결혼기념일 / 생일선물 / 레이어드]
```
**노려야 할 검색어**: 금목걸이, 순금목걸이, 골드목걸이, 14k목걸이

### 팔찌·발찌 (cate_no=28)
```
소아벨라 [24K 순금 / 14K / 18K] 팔찌 [디자인명] — [선물 / 커플 팔찌 / 기념일]
```
**노려야 할 검색어**: 금팔찌, 순금팔찌, 골드팔찌, 커플팔찌

### 귀걸이 (cate_no=26)
```
소아벨라 [24K 순금 / 14K / 18K] 귀걸이 [스터드 / 후프 / 드롭] — [일상 / 웨딩 / 선물]
```
**노려야 할 검색어**: 금귀걸이, 순금귀걸이, 14k귀걸이, 골드귀걸이

## Cafe24 일괄 적용 방법

Cafe24는 상품을 개별 편집할 수 있지만, 상품이 많을 경우:

1. **Cafe24 상품 일괄 등록 Excel**: 관리자 → 상품 관리 → 상품 일괄 등록/수정
   - 엑셀 다운로드 → `이미지 alt` 컬럼 채우기 → 일괄 업로드
2. **스킨 변수 활용 (반자동)**: `product/detail.html` 스킨에서 이미지 태그 수정
   ```html
   <!-- 수정 전 -->
   <img src="{$product_image}">

   <!-- 수정 후: 상품명을 alt로 자동 사용 -->
   <img src="{$product_image}" alt="소아벨라 {$product_name}">
   ```
   → 완벽하진 않지만 alt 없는 것보다 훨씬 낫고, 즉시 적용 가능

## 주의사항

- alt에 키워드를 **억지로 나열하지 말 것** (스팸 패널티)
  - 나쁜 예: `금반지 순금반지 금반지추천 돌반지 싼금반지 금반지쇼핑몰`
  - 좋은 예: `소아벨라 24K 순금반지 하트 디자인 — 돌잔치 선물`
- **125자 이내** 권장
- 장식용 이미지(구분선, 배경)는 `alt=""` 빈 값이 정상
