<h1> 태그 없음 — 검색엔진이 이 페이지의 핵심 주제를 파악할 수 없음

<h2>소아벨라 | 한국공식금거래소</h2>

meta description 없음 — 구글이 임의의 텍스트를 발췌해 검색 결과에 표시

클릭률(CTR) 직접 영향. 구매 유도 문구 없이 크롤러가 무작위 텍스트 사용 중.

BEFORE (현재)
<!-- meta description 없음 -->
AFTER (수정)
<meta name="description"
  content="24K 순금 반지·목걸이·팔찌
  전문 쇼핑몰 소아벨라. 한국공식
  금거래소 인증, 정품 보증서 발급.
  전국 무료배송.">
예상 효과: CTR +20~30% 향상
CRITICAL

canonical 태그 없음 — ?srsltid= 파라미터 URL이 별도 페이지로 색인 중

유입 URL 예: soavela.com/?srsltid=AfmBOop... → 홈과 동일 콘텐츠인데 다른 페이지로 분류
→ 동일 페이지가 여러 URL로 쪼개져 순위 점수가 분산됨.

BEFORE (현재)
<!-- canonical 없음 -->
soavela.com/?srsltid=AfmBOop...
→ 별도 페이지로 색인됨
AFTER (수정)
<link rel="canonical"
  href="https://soavela.com/">
→ 모든 파라미터 URL이
  홈으로 통합됨
예상 효과: 중복 색인 제거 순위 점수 통합
CRITICAL

Open Graph 태그 없음 — 카카오톡·SNS 공유 시 빈 화면 표시

링크를 공유해도 이미지·제목·설명 없이 URL만 표시 → 클릭 유도 불가.

robots.txt Sitemap 선언 위치 오류

Sitemap: https://soavela.com/sitemap.xml이 bingbot 섹션에만 존재.
Googlebot·네이버봇은 robots.txt에서 sitemap 경로를 찾지 못함.
→ Google Search Console + 네이버 서치어드바이저에 직접 수동 제출로 해결.

8주 구현 로드맵
Week 1 즉시 효과

Cafe24 스킨 메타태그 적용

  • 홈 title / meta description / canonical 삽입
  • 카테고리 7개 개별 메타태그 분기 처리
  • Open Graph + Twitter Card 전 페이지 적용
  • 이미지 alt 텍스트 스킨 변수 자동화
→ SNS 공유 카드 정상화 + 색인 오류 감소
Week 2 리치 스니펫

JSON-LD 구조화 데이터 적용

  • 상품 페이지: Product + BreadcrumbList + AggregateRating
  • 홈페이지: Organization + WebSite + ItemList
  • Google Search Console 등록 + sitemap 제출
  • 네이버 서치어드바이저 등록 + 수집 요청
→ 구글 쇼핑 탭 노출 + 별점·가격 리치 스니펫 준비
Week 3~4 콘텐츠

롱테일 키워드 정보성 콘텐츠 4편

  • 「순금 24K·18K·14K 차이 완전 정리」
  • 「돌반지 완벽 가이드 — 돈수·가격 계산법」
  • 「여자친구 생일 금목걸이 선물 예산별 추천」
  • 「커플링 순금 vs 14K 선택 기준」
→ 롱테일 키워드 색인 시작 + 이미지 검색 유입 증가
Week 5~8 확장

FAQ 콘텐츠 + 로컬 SEO + 성과 측정

  • 「금 한 돈 몇 그램?」 FAQ JSON-LD 적용
  • 구글 비즈니스 프로필 등록 (종로구 오프라인 위치)
  • Search Console 검색어 리포트 분석 후 전략 조정
  • Core Web Vitals (PageSpeed) 점수 측정 + 이미지 WebP 전환
→ "금반지" "금목걸이" 메인 키워드 순위 상승 시작
