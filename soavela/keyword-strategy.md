## 3단계 키워드 포트폴리오

### 1단계 — 브랜드 키워드 (즉시 1위 가능)
> 이미 소아벨라를 알고 있는 사람이 검색

| 키워드 | 목표 | 대응 페이지 |
|--------|------|------------|
| 소아벨라 | 브랜드 1위 유지 | 홈페이지 |
| 소아벨라 반지 | 카테고리 직접 유입 | 반지 카테고리 |
| 소아벨라 목걸이 | 카테고리 직접 유입 | 목걸이 카테고리 |
| 벨라골드 | 법인명 검색 | 홈페이지 |

### 2단계 — 롱테일 키워드 (1~3개월 내 진입 목표)
> 검색량은 적지만 구매 의도가 명확한 키워드

| 키워드 | 월 예상 검색량 | 경쟁도 | 타겟 콘텐츠 |
|--------|-------------|--------|------------|
| 24k 순금반지 돌잔치 | 낮음 | 낮음 | 블로그 글 |
| 순금반지 선물 추천 | 중간 | 중간 | 카테고리 최적화 |
| 14k 18k 반지 차이 | 중간 | 낮음 | 정보성 블로그 글 |
| 순금 한 돈 가격 | 높음 | 중간 | FAQ 페이지 |
| 금목걸이 레이어드 | 낮음 | 낮음 | 상품 상세 최적화 |
| 커플링 순금 14k 차이 | 낮음 | 낮음 | 블로그 글 |
| 금팔찌 선물 여자친구 | 낮음 | 낮음 | 블로그 글 |
| 종로 금반지 | 중간 | 낮음 | 로컬 SEO |

### 3단계 — 메인 키워드 (3~6개월 목표)
> 롱테일 콘텐츠가 쌓이면 자연스럽게 권위가 올라가는 키워드

| 키워드 | 월 예상 검색량 | 목표 순위 |
|--------|-------------|----------|
| 금반지 | 매우 높음 | TOP 10 |
| 금목걸이 | 매우 높음 | TOP 10 |
| 순금반지 | 높음 | TOP 5 |
| 순금목걸이 | 높음 | TOP 5 |
| 돌반지 | 높음 | TOP 3 |
| 커플링 | 높음 | TOP 10 |

## 콘텐츠 캘린더 (8주)

> Cafe24 게시판/블로그 기능 또는 별도 서브도메인 `blog.soavela.com` 활용

### Week 1~2: 제품 정보형 콘텐츠
> 목적: 기초 색인 + "전문 쇼핑몰" 신뢰도 확보

**글 1: 「순금 24K, 18K, 14K 차이 완전 정리 — 뭘 사야 할까?」**
- 타겟 키워드: `순금 14k 18k 차이`, `금 캐럿 차이`
- 구조:
  - 24K = 순도 99.9% (순금), 가장 비쌈, 변색 없음
  - 18K = 순도 75%, 일상 착용 내구성 좋음
  - 14K = 순도 58.5%, 가성비, 다양한 색상
  - 소아벨라 추천: 용도별 선택 가이드
- 예상 분량: 800~1,000자 + 비교 표

**글 2: 「돌반지 완벽 가이드 — 순금 돌반지 고르는 법과 돈수 계산」**
- 타겟 키워드: `돌반지`, `돌반지 몇 돈`, `순금 돌잔치 선물`
- 구조:
  - 돌반지란? (전통적 의미)
  - 몇 돈이 적당한가? (1돈 = 3.75g 기준)
  - 순금 vs 18K 돌반지 차이
  - 소아벨라 돌반지 추천 상품 3선 + 직접 링크

### Week 3~4: 선물 가이드형 콘텐츠
> 목적: 거래형 검색 의도 공략 + 시즌 트래픽 확보

**글 3: 「여자친구 생일 금목걸이 선물 — 예산별 추천 5선」**
- 타겟 키워드: `금목걸이 선물`, `여자친구 생일 선물 금`, `30대 여자 금목걸이`
- 구조:
  - 5만원대 / 10만원대 / 30만원대 / 50만원+ 구간별 추천
  - 소아벨라 상품 직접 링크 포함
  - "처음 금 선물하는 사람을 위한 팁" 섹션

**글 4: 「커플링 순금 vs 14K — 결혼 10년차가 추천하는 선택 기준」**
- 타겟 키워드: `커플링`, `커플링 추천`, `순금 커플링`
- 구조:
  - 가격 차이 실제 비교
  - 내구성·변색 차이 직접 사용 경험
  - 소아벨라 커플링 베스트 3선

### Week 5~6: FAQ + 금 시세 활용 콘텐츠
> 목적: 구글 FAQ 리치 스니펫 + 반복 방문 유도

**글 5: 「금 한 돈 몇 그램? 순금 무게 단위 완전 정리」**
- 타겟 키워드: `금 한 돈`, `금 돈 그램`, `순금 한 돈 가격`
- FAQ 구조화 데이터(JSON-LD) 필수 적용
- 구조:
  - 1돈 = 3.75g (이걸 모르는 사람이 많음)
  - 돌반지 1돈짜리 가격 계산 방법
  - 오늘 금 시세 확인 방법 (한국금거래소 링크)

**글 6: 「금목걸이 레이어드 코디 방법 — 굵기·길이별 조합 가이드」**
- 타겟 키워드: `금목걸이 레이어드`, `목걸이 레이어드 코디`
- 비주얼 중심 콘텐츠 (이미지 SEO 강화)
- 소아벨라 상품 착용 사진 + alt 텍스트 최적화

### Week 7~8: 로컬 SEO + 신뢰도 콘텐츠
> 목적: "종로 금반지" 류 지역 검색 공략

**글 7: 「한국공식금거래소 인증이란? — 순금 구매 전 꼭 확인해야 할 것」**
- 타겟 키워드: `한국공식금거래소`, `순금 정품 확인`, `순금 보증서`
- 소아벨라의 신뢰도 증명 콘텐츠
- E-E-A-T(전문성·권위·신뢰) 점수 강화

**글 8: 「종로구 금 주얼리 — 온라인으로 더 싸게 사는 방법」**
- 타겟 키워드: `종로 금반지`, `종로 귀금속`
- 로컬 비즈니스 스키마와 연계

## 네이버 블로그 병행 전략

구글 SEO와 별도로 **네이버 블로그**를 운영하면 국내 검색에서 훨씬 빠른 효과:

| 채널 | 특징 | 목표 키워드 |
|------|------|------------|
| 네이버 블로그 | C-Rank + DIA 알고리즘, 빠른 색인 | 돌반지, 금선물 추천, 순금팔찌 |
| 소아벨라 공식 웹 | 구글 장기 권위 축적 | 금반지, 금목걸이, 24k순금 |
| 인스타그램 | 이미지 검색 + 브랜드 인지도 | (검색 엔진 직접 영향 없음) |
